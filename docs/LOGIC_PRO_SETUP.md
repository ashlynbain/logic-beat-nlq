# Logic Pro setup

Each session also writes `output/<session_id>/LOGIC_SETUP.txt`.

## Tempo

Set project tempo to the BPM shown in the app. If Logic asks on import, use **MIDI file tempo**.

## Loop

1. Cycle region over **bars 1–N** (N from the app, ~2 minutes).
2. Press **C** for cycle mode.

## Instruments

Assign plugins per track (MIDI has no built-in sounds):

| MIDI track | Plugin |
|------------|--------|
| Drums | Drum Kit Designer or Drum Machine Designer |
| Bass / 808 Bass | Studio Bass |
| Keys | Vintage Electric Piano |
| Synth | Retro Synth |

## Vocals

Add an audio track and record over the loop.
